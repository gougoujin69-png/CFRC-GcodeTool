# CFRC-GcodeTool

**连续纤维增强复合材料 (CFRC) 3D打印 G-code 生成工具集**

面向连续纤维增强复合材料（Continuous Fiber Reinforced Composites）增材制造场景，提供两个互补的 G-code 生成工具：一个功能完整的 **Python GUI 桌面应用**，以及一个面向快速批量导出的 **Excel VBA 插件**。

---

## 工具总览

| 特性 | Python GUI (`PathCutGcode_GUI_v5`) | Excel 插件 (`CFRC_GcodeExporter`) |
|---|---|---|
| **适用场景** | 多路径编辑、复杂剪丝流程、精细参数调试 | 从坐标表快速批量导出 G-code |
| **数据输入** | GUI 手动输入 / CSV / JSON 工程文件 | Excel 表格 (A=X, B=Y, C=Z, D=Cut) |
| **挤出模式** | 绝对 E (M82) / 相对 E (M83) 可选 | 相对 E (M83) |
| **剪丝逻辑** | 完整 O→A→B→C→D→E 六点剪丝流程 | 到达剪丝点 → 宏命令 → E 重置 → 空驶 |
| **坐标变换** | 缩放、偏移、Y 翻转 | 直接使用原始坐标 |
| **Header** | 自动生成 G90/G21/M82/M83 | 完全自定义（支持 OrcaSlicer 风格） |
| **平台** | Windows / macOS / Linux (Python 3.8+) | Windows (Excel 2016+) |
| **依赖** | Python 标准库 (tkinter) | 无（纯 VBA） |

## 1. Python GUI — `PathCutGcode_GUI_v5`

全功能桌面应用程序，专为 CFRC 打印中需要**多路径管理**和**精确剪丝时序控制**的场景设计。

### 核心功能

- **多路径管理**：每条路径独立命名、独立 Z 层高，支持排序、增删
- **点编辑**：逐点添加/插入/删除/排序，支持从文本批量粘贴
- **六点剪丝流程 (O→A→B→C→D→E)**：

```
路径1打印方向 ────────────►
            O              A        B (安全Z)
            ├──打印(有E)──►├─(不带E)─►├── 抬升
            │              │          │
            │  ┌ 剪丝宏    │          │ 空移到C
            │  └──────────►│          ▼
                                     C ── FEED65 (快速挤出宏)
                                     │
                                     ▼ 下降到D (略低于Z2)
                                     D ── 暂停(可配)
                                     │
                                     ▼ 慢速打印 D→E
                                     E ── 下一路径起点
```

- **坐标变换**：缩放系数、X/Y 偏移、Y 轴翻转（适配不同打印机坐标系）
- **挤出控制**：可选绝对 E (M82) 或相对 E (M83)，可配 E/mm 系数
- **工程管理**：JSON 工程文件保存/加载，CSV 导入/导出
- **G-code 预览**：生成前预览完整 G-code

### 快速启动

```bash
# 无需安装任何第三方库
cd gui
python PathCutGcode_GUI_v5.py
```

> 需要 Python 3.8+ 和 tkinter（Python 自带，通常无需额外安装）。

### 使用流程

1. 启动程序 → 点击 **新建路径** → 设置路径名和 Z 层高
2. 在点编辑面板中添加路径点坐标（手动输入或"从文本粘贴"）
3. 在右侧面板配置：进给速度、E 挤出参数、剪丝流程参数
4. 点击 **预览G-code** 检查输出
5. 点击 **生成G-code并保存** 导出 `.gcode` 文件

### CSV 导入格式

支持三种 CSV 格式自动识别：

```
# 2列 (x, y) — 全部归入一条路径
50.0, 50.0
150.0, 50.0

# 3列 (路径名, x, y) — 按名称分组
Layer1, 50.0, 50.0
Layer1, 150.0, 50.0
Layer2, 200.0, 50.0

# 4列 (路径名, x, y, z) — 按名称分组 + 独立Z
Layer1, 50.0, 50.0, 0.20
Layer1, 150.0, 50.0, 0.20
Layer2, 200.0, 50.0, 0.40
```

### 剪丝参数说明

| 参数 | 含义 | 默认值 |
|---|---|---|
| `O←A 距离 L` | 路径终点 A 回退到剪丝触发点 O 的距离 | 15 mm |
| `O 点宏` | 剪丝时执行的宏命令（支持多行） | `M280` |
| `抬升 ΔZ` | A 到安全点 B 的 Z 抬升量 | 3 mm |
| `C 距离` | 下一路径起点 E 反向到快速挤出点 C 的距离 | 10 mm |
| `D 距离` | E 反向到逼近点 D 的距离 | 2 mm |
| `C 处额外量` | FEED65 宏的 E 同步量 | 0.6 mm |
| `C→D 逼近速度` | C 到 D 的进给速度 | 3600 mm/min |
| `D 点低于 Z` | D 点低于下一层 Z 的偏移 | 0.05 mm |
| `D 点暂停` | D 点滞留时间 | 500 ms |
| `D→E 慢速打印速度` | 从 D 缓慢进入 E（下一路径起点）的速度 | 600 mm/min |

## 2. Excel 插件 — `CFRC_GcodeExporter`

面向**快速批量导出**场景的 Excel VBA 加载项。当你的路径坐标已经在 Excel 表格中（例如从 MATLAB 导出），直接点一下按钮即可生成 G-code。

### 核心功能

- **一键导出**：从 Excel 活动工作表直接导出 `.gcode` 文件
- **自定义 Header**：支持粘贴完整的 OrcaSlicer 风格 G-code 头
- **自动挤出计算**：`E = 段长 × 线宽 × 层高 / (π·(丝径/2)²)`，相对模式 (M83)
- **剪丝标记**：在 D 列写 `1` 即可在该点插入剪丝宏命令
- **自动换层识别**：Z 突变 > 阈值时自动空驶
- **参数记忆**：所有工艺参数自动保存，下次打开 Excel 默认复用

### 数据格式

| A列: X (mm) | B列: Y (mm) | C列: Z (mm) | D列: Cut (1=剪丝) |
|---|---|---|---|
| 50 | 50 | 0.2 | |
| 150 | 50 | 0.2 | |
| 150 | 150 | 0.2 | |
| 50 | 50 | 0.2 | **1** |
| 200 | 50 | 0.2 | |
| 300 | 50 | 0.2 | |

- D 列写 `1`、`Y`、`YES`、`TRUE`、`CUT` 均视为剪丝
- 其余点 D 列留空即可
- 第一行可以是表头（导出时会询问）

### 安装步骤（仅需一次）

1. 打开 Excel → `文件` → `选项` → `自定义功能区` → 勾选 **开发工具**
2. 新建空白工作簿 → **Alt+F11** 打开 VBA 编辑器
3. 右键 `VBAProject` → **导入文件** → 选 `excel_plugin/CFRC_GcodeExporter.bas`
4. 关闭 VBA 编辑器 → `文件` → `另存为` → 类型选 **Excel 加载项 (*.xlam)** → 保存
5. `开发工具` → `Excel 加载项` → 勾选 **Cfrc_Gcodeexporter**
6. （可选）右键快速访问栏 → 自定义 → 添加宏 `ExportGcodeFromSheet`

> 详细图文说明见 [`docs/Excel_Plugin_Install_Guide.md`](docs/Excel_Plugin_Install_Guide.md)

### 使用方法

1. 打开包含路径数据的 xlsx 文件
2. 点击快速访问栏的"导出 G-code"按钮（或 **Alt+F8** → `ExportGcodeFromSheet`）
3. 确认工艺参数 → 剪丝宏命令 → Header → 是否有表头
4. 选择保存路径 → 完成

### 输出示例

```gcode
; HEADER_BLOCK_START
; generated by OrcaSlicer 2.2.0 on 2026-04-15 at 15:15:01
; ...
M83                         ; 挤出机相对模式
G92 E0                      ; 重置挤出机

; ===== Toolpath start (from XLSX) =====
G92 E0                      ; 重置挤出
G1 X50 Y50 Z0.2 F3000 ; 移动到起点
G1 X150 Y50 E1.6611 F1800
G1 X150 Y150 E1.6611 F1800
G1 X50 Y150 E1.6611 F1800
G1 X50 Y50 E1.6611 F1800
; ---- Cut at point 5 ----
CUT_FIBER
G92 E0                      ; 剪丝后重置 E
G1 X200 Y50 Z0.2 F3000 ; travel after cut
...
```

## 挤出量计算原理

两个工具使用相同的体积守恒挤出模型：

**相对模式 (M83)**：每段挤出量 = 段长 × 截面面积比

```
E = L × (W × H) / (π × (D/2)²)
```

其中：
- `L` = 两点间距离 (mm)
- `W` = 线宽 (mm)，默认 0.4
- `H` = 层高 (mm)，默认 0.2
- `D` = 丝材直径 (mm)，默认 1.75

以默认参数为例：`e_per_mm = 0.4 × 0.2 / (π × 0.875²) ≈ 0.0332 mm/mm`

Python GUI 同时支持**绝对模式 (M82)**，E 值累加并在剪丝后通过 `G92 E0` 重置。
